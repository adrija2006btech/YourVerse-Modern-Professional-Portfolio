
# 🌌 YourVerse | Professional Digital Portfolio

A sophisticated, multi-page personal portfolio and interactive resume built with a **Warm Earth & Slate** aesthetic. Designed for high readability and a "Resume-Ready" professional presence.

## ✨ Design Philosophy
Unlike typical high-contrast dark modes, **YourVerse** utilizes a muted palette of **Deep Charcoal (#1A1A1C)** and **Terracotta (#E29578)**. This approach reduces eye strain while maintaining a high-end, editorial feel that prioritizes content and typography.

## 🚀 Key Features
- **Multi-Page Architecture:** Seamless navigation between Home, About, Projects, Skills, and Contact.
- **Interactive Resume Elements:**
    - **Live Stats Grid:** Showcasing project counts and technical contributions.
    - **Experience Timeline:** A chronological view of professional growth.
- **Responsive Navigation:** Sophisticated desktop sidebar that transforms into a mobile-friendly bottom bar.
- **Optimized UI/UX:** Built with modern CSS Grid and Flexbox for perfect alignment across all screen sizes.
- **Performance Focused:** Lightweight code with zero external heavy libraries.

## 🛠️ Tech Stack
* **Structure:** HTML5 (Semantic & SEO Friendly)
* **Styling:** CSS3 (Custom Properties, Grid, Flexbox, Keyframe Animations)
* **Logic:** JavaScript (ES6+ for interactive navigation and UI states)
* **Data Background:** MySQL (Database architecture logic for project management)
* **Design:** UI/UX principles focused on "Subtle Warmth" and minimalism.
